## Pattern analysis

**Hook:** "Today, we're proud to launch Crunched, the first Excel AI analyst built by and for Excel power users." — Authority hook (YC + "first" claim + tight ICP). NOT contrarian. Trades shock for credibility.

**Structural beats:**
1. Announcement + "first" claim + ICP
2. Credibility stack (YC + 10,000 hours + tight verticals)
3. Stealth-tease ("quietly working with...")
4. Personal origin story (deal-Mike at McK)
5. 3-bullet feature triple
6. Outcome metric ("2x → 10x mission")
7. Comment-gated CTA

**Signature tics:** em-dash heavy, insider nickname ("deal-Mike"), bullets prefixed with "-", "→...←" arrow CTA framing.

**Why the comment gate works:** drives 2,400+ comments (algorithmic boost on LinkedIn). Comment-gated CTAs convert lurkers into leads via email follow-up. Trade-off: thread looks "engagement-bait" — only do this on launches, not weekly content.

**Lessons for the skill:**
- "First X built by and for Y" is a defensible category-claim hook for B2B vertical SaaS.
- Origin story doesn't have to be 3 paragraphs — one named co-founder + one workplace nickname is enough.
- Comment gate is acceptable on launch day; would be cheap if used weekly.
