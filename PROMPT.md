# The Prompt Pack

Five prompts. Drop into Claude or ChatGPT. They encode my 5-beat formula, the 8 hard rules, and the 6 archetype patterns.

Use Prompt 1 for the first draft. Use Prompt 5 to audit it before you ship.

## Prompt 1 — The Master Launch Post Generator

**Use when:** You have a product to launch and want a draft in 60 seconds.

```
You are a launch-post writer trained on 30+ viral SaaS launch posts (Lovable, Devin, Bolt.new, Cluely, Icon, Crunched, Snowglobe, Deel, Gamma, Supabase, etc.).

Write a launch post for the product below using the 5-beat formula from Renat Gabitov / Represent Studio.

PRODUCT DETAILS:
- Name: [PRODUCT NAME]
- Category: [e.g., "AI sales assistant", "Excel AI analyst", "no-code app builder"]
- One-line value prop: [WHAT IT DOES IN ONE LINE]
- Problem it solves: [THE PAIN]
- How it works: [3-4 STEPS]
- Key proof points: [METRICS, CUSTOMERS, INVESTORS, COMPARABLES]
- Founder name: [NAME]
- Founder background: [1-LINE CREDENTIAL]
- Video: [YES/NO — and if yes, the video's opening line]
- CTA goal: [SIGNUPS / DISCOVERY CALLS / WAITLIST / LEAD MAGNET / BRAND]
- Platform: [X / LINKEDIN / BOTH]
- Length: [SINGLE TWEET / LONG TWEET / THREAD]

Apply these 5 beats:
1. CLEAR ANNOUNCEMENT — "Introducing [Product], the [world's first / best] [Category]." No throat-clearing.
2. PROBLEM STATEMENT — one sentence + adjective that "twists the knife" (e.g., "brutally slow", "quietly expensive")
3. SOLUTION — one sentence + quality guard ("without [bad thing]")
4. HOW IT WORKS — 3-4 steps in arrows (→) or numbers
5. CTA — pick ONE: engagement tactic ("Comment X for Y") OR direct link (in reply, NOT main post)

Apply these HARD RULES:
- NO em dashes (use periods + line breaks)
- NO "excited to share" / "stay tuned" / throat-clearing
- NO link in main post (link goes in reply tweet)
- NO stacked CTAs (pick one)
- Match the video hook if there is one
- Founder voice ("I" / "we"), not company-account voice
- Use ONE analogy bridge if the product is in a new category (like "ChatGPT + CapCut for ads")
- Include ONE specific human detail (city, time, nickname, customer first name)

DELIVER:
- The main post (formatted for the chosen platform/length)
- The reply tweet (with the link)
- 3 alternative hooks for A/B testing
- A 1-line note on why this hook should work

Now write the launch post.
```

## Prompt 2 — The Hook-Sharpener

**Use when:** You have a draft post but the hook isn't landing.

```
I have a launch-post hook. Run it through this diagnostic and rewrite if needed.

CURRENT HOOK:
"[PASTE YOUR FIRST SENTENCE]"

PRODUCT CONTEXT:
[1-2 sentences about what you're launching]

DIAGNOSTIC:
1. First-7-words test — do the first 7 words contain a number, name, or specific noun? (Generic openers like "We're excited to..." fail this.)
2. Strip-3 test — strip the first 3 words. Does the rest still grab? If yes, those 3 words were filler.
3. Commodity test — could a competitor post the same hook? If yes, sharpen with a number, name, or constraint.
4. Tweet-2 earn test — does the hook earn the next sentence? Will the reader want more?
5. Tone-match test — does the hook tone match what the product/video actually delivers?

Then rewrite the hook in 5 different patterns:
- Numbered-proof hook ("I analyzed [N] [things]. [Bold claim].")
- Inversion hook ([X] will [Y] — flip an expected relationship)
- Distance/constraint hook (specific number + unlikely constraint)
- Question hook (a question the reader needs answered)
- Result-first hook ("[Specific outcome]. Here's how.")

For each, give the hook + a 1-line note on when to use it.
```

## Prompt 3 — The Archetype Picker

**Use when:** You're not sure which format to use.

```
Help me pick the right launch-post archetype for my situation.

MY SITUATION:
- Product: [WHAT YOU'RE LAUNCHING]
- Stage: [PRE-LAUNCH WAITLIST / DAY-OF LAUNCH / FUNDRAISE / MILESTONE / FREE RESOURCE]
- Audience size: [< 1K / 1K-10K / 10K+]
- Story: [ANY UNUSUAL ORIGIN, CONSTRAINT, OR PIVOT? Y/N — describe]
- Video status: [HAVE A KILLER 30-90s VIDEO / HAVE A LONGER BRAND FILM / NO VIDEO]
- Goal: [PIPELINE / WAITLIST SIGNUPS / BRAND / FUNDRAISE BUZZ / FREE RESOURCE FUNNEL]

Pick ONE of these 6 archetypes and tell me why:

1. **YC-style single tweet** — when product self-explains, founder has audience, video does the work
2. **Origin-story thread** — when story > feature, first-time founder, unusual journey
3. **Teardown / reveal thread** — when "how we built it" is as interesting as "what we built"
4. **Fundraise thread** — when raise is the moment + you want to tell the full origin story
5. **Lead-magnet + service upsell** — when you're an agency / expert giving away a free thing to funnel to a paid service
6. **Agency amplification** — when you're writing about a CLIENT's launch, not your own

Then tell me:
- Which archetype, and why
- Which length to use (single tweet / long tweet / thread)
- Which template to fill in
- 1 risk to watch for with this archetype
```

## Prompt 4 — The Multi-Format Multiplier

**Use when:** You have ONE launch and want to ship it as multiple posts across days/platforms.

```
I have a launch and I want to multiply it into 4 distinct posts so I'm not wasting the asset.

LAUNCH CONTEXT:
- Product: [NAME + ONE-LINE]
- The launch video: [DESCRIPTION + DURATION]
- Key facts: [3-5 BULLETS — investors, customers, metrics, unusual story]
- Founder voice samples: [PASTE 1-2 EXISTING POSTS BY THIS FOUNDER]

Generate FOUR distinct posts from the same launch:

POST 1: Day-of launch — single tweet on X, video embedded, hook + 3 lines.
POST 2: Day 1 — origin-story thread on X (8-12 tweets) telling the founder journey.
POST 3: Day 2 — long post on LinkedIn (300-500 words) for the LinkedIn audience.
POST 4: Day 7 — milestone follow-up post (a customer win, a metric, a press hit).

For each:
- Different hook (don't repeat the first line)
- Different angle (one tactical, one emotional, one social-proof, one milestone)
- Same underlying facts
- Each ends with ONE CTA appropriate to the post

Apply Renat's 5-beat formula and the 8 hard rules to each.
```

## Prompt 5 — The Pre-Ship Auditor

**Use when:** You have a draft and want a final review before publishing.

```
Audit this launch post draft for shippability.

DRAFT:
[PASTE YOUR FULL DRAFT POST]

PLATFORM: [X / LINKEDIN]
ARCHETYPE: [WHICH OF THE 6]
VIDEO: [YES + opening line / NO]

Run this 12-point checklist and give me a PASS / FAIL on each, with a 1-line fix where it fails:

1. ✅/❌ Opens with product name + category in first 10 words
2. ✅/❌ First 7 words contain a number, name, or specific noun
3. ✅/❌ Problem statement uses an adjective that "twists the knife"
4. ✅/❌ One-sentence solution with quality guard ("without [bad thing]")
5. ✅/❌ 3-4 step "how it works" in arrows or numbers (terse, not wordy)
6. ✅/❌ ONE CTA (not stacked)
7. ✅/❌ NO link in main post (link is in reply / comment)
8. ✅/❌ NO em dashes anywhere
9. ✅/❌ NO "excited to share" / "stay tuned" / throat-clearing
10. ✅/❌ Post hook matches video hook (if video exists)
11. ✅/❌ Founder voice (I/we), not corporate
12. ✅/❌ Closing line is either a quotable thesis OR a single specific action

Then output the fixed version with all FAIL points addressed.
```
